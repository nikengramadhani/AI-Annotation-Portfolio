# Identity Dataset

## Overview

Worked on computer vision datasets involving identity documents and hotel key cards. The project focused on accurately identifying and annotating objects while following strict privacy and confidentiality requirements.

## Responsibilities

- ID card annotation
- Hotel key card annotation
- Object localization
- Annotation quality review
- Dataset validation

## Skills Used

- Image Annotation
- Data Validation
- Quality Assurance
- Attention to Detail
- Privacy Awareness

## Note

Project details and datasets cannot be shared publicly due to confidentiality agreements. All annotation work involving identity-related objects was performed following client privacy and data protection requirements.
