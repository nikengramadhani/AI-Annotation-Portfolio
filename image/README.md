# Images

This folder contains sample illustrations used throughout this portfolio to demonstrate AI data annotation workflows and object labeling techniques.

The images are recreated examples using publicly available or self-created visuals for demonstration purposes only. No confidential client data, proprietary datasets, or project-specific images are included in this repository.

These illustrations represent the types of annotation tasks I have worked on, including:

- Vehicle annotation
- License plate annotation
- Car logo annotation
- Telecommunication tower annotation
- People detection and counting
- Identity object annotation
- Maritime crew annotation

The purpose of these images is to provide a visual representation of my experience while respecting confidentiality agreements and data privacy requirements.
